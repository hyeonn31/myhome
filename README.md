## 개인 홈페이지 디자인
3D 모델링 작업물과 “정승현”이라는 디자이너를 소개할 수 있는 개인 웹페이지를 제작했습니다. 포트폴리오를 간단히 블럭화하여 구분하고 그 블럭을 클릭하면 작품을 자세히 볼 수 있도록 했습니다. 
Home, Portfolio, About, Contact로 페이지가 구성되어 있습니다.
